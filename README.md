# license-plate-detection

Overview
A computer vision-based system to detect vehicle license plates from images using OpenCV and OCR.

Features
- Image preprocessing
- Edge detection and contour analysis
- License plate text extraction using OCR

Technologies Used
- Python
- OpenCV
- Tesseract OCR

How It Works
The system processes an input image, detects rectangular regions resembling license plates,
and extracts text using OCR techniques.

How to Run
1. Install dependencies using `pip install -r requirements.txt`
2. Place a car image inside `sample_images/`
3. Run `python main.py`
